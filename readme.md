# Java Notepad

A simple Notepad application built with Java Swing.

## Features

- Open and save text files
- Keyboard shortcuts:
  - Ctrl+O: Open file
  - Ctrl+S: Save file
  - Ctrl+F4: Close app

## How to Run

1. Compile:
    ```
    javac src/Notepad.java
    ```
2. Run:
    ```
    java src Notepad
    ```
